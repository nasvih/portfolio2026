# Muhammed Nasvih V — Portfolio

Personal portfolio of **Muhammed Nasvih V**, Lead DevOps Engineer.
Single-page site: cloud · CI/CD · Kubernetes · security · automation.

## Run locally
Static site — no build step.
```bash
python3 -m http.server 4174   # then open http://localhost:4174
```

## Structure
- `index.html` — the whole site (HTML + CSS + a little JS)
- `public/profile-photo.jpg` — profile image

## Deploy
Host the folder on any static host (GitHub Pages, Netlify, Cloudflare Pages,
Hostinger, S3 + CloudFront). Point the domain at it.
